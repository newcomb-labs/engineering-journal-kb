# Newcomb Labs Engineering Journal

Engineering knowledge base, learning journal, and documentation portal for labs, systems, and cross-domain study.

## Purpose

This repository serves as a structured engineering knowledge system combining:

- Documentation
- Zettelkasten-style knowledge notes
- Structured lab reports
- Architecture and runbooks
- Cross-domain study

The system is designed to support long-term learning, documentation discipline, and reproducible engineering workflows.

## Structure

docs/

- notes/ — staging notes from classes, reading, and experiments
- zettel/ — refined permanent knowledge notes
- mocs/ — maps of content (topic indexes)
- labs/ — structured lab reports
- runbooks/ — operational procedures
- architecture/ — system design documentation
- incidents/ — incident analysis and lessons learned
- diagrams/ — reusable diagrams
- templates/ — documentation templates

## Documentation System

Documentation is built using:

- MkDocs Material
- Mermaid diagrams
- Wiki-style links
- Backlinks
- Tag indexing

## Workflow

Notes typically follow this pipeline:

notes → refinement → zettel → linked knowledge graph

Labs and operational knowledge are documented separately using structured templates.
